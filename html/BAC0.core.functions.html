BAC0.core.functions.discoverPoints module¶

This module define discoverPoints function

BAC0.core.functions.discoverPoints.discoverPoints(bacnetapp, address, devID)¶

This function allows the discovery of all bacnet points in a device

Parameters:
  • bacnetApp – The app itself so we can call read
  • address – address of the device as a string (ex. ‘2:5’)
  • devID – device ID of the bacnet device as a string (ex. ‘1001’)
Returns:

a tuple with deviceName, pss, objList, df

deviceName : name of the device pss : protocole service supported objList : list of bacnet object (ex. analogInput, 1) df : is a dataFrame containing pointType, pointAddress, pointName, description presentValue and units

If pandas can’t be found, df will be a simple array
